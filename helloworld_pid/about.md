# PID 入门与单自由度平衡球仿真（从零理解版）

---

# 一、这个项目到底在做什么？

我们要做的是一个经典控制问题：

👉 在一块可以转动的板子上放一个小球，让它始终停在中间

你可以理解为：

* 控制输入：板子的角度 θ
* 被控制对象：小球的位置 x

目标：

👉 让 x → 0（始终在中心）

---

# 二、核心物理模型（非常重要）

在简化情况下：

x'' = g · θ

含义：

* θ（板子角度）决定小球的加速度
* 角度越大，小球加速越快

⚠️ 关键理解：

👉 你控制的不是“位置”，而是“加速度”

这就是系统难点所在

---

# 三、为什么不能直接用一个 PID？

如果你直接做：

位置误差 → PID → 电机

会出现：

* 反应慢
* 大幅震荡
* 很难稳定

原因：

👉 系统是“二阶”的（控制加速度）

---

# 四、正确方法：级联控制（外环 + 内环）

我们把问题拆成两层：

## 外环（慢）：控制位置

输入：位置误差
输出：目标角度 θ_ref

👉 “球偏左 → 板子向左倾一点”

---

## 内环（快）：控制角度

输入：角度误差
输出：电机控制量

👉 “让板子快速达到目标角度”

---

# 五、完整控制结构

x_target → [位置PID] → θ_ref → [角度PID] → 电机 → θ → 小球 → x

👉 两个 PID 串联（级联控制）

---

# 六、代码逐行讲解

下面我们结合代码逐块理解

---

## 1. 导入库

import numpy as np

## 2. 系统参数

```python
g = 9.81
dt = 0.01
T = 10.0
```

解释：

* g：重力加速度
* dt：每一步时间（越小越精确）
* T：总仿真时间

---

## 3. PID 控制器类

```python
class PID:
```

这是一个通用 PID 控制器

---

### 初始化

```python
def __init__(self, kp, ki, kd):
```

三个参数：

* kp：比例（当前误差）
* ki：积分（累计误差）
* kd：微分（变化速度）

---

### 更新函数

```python
def update(self, error, dt):
```

输入：

* error：当前误差
* dt：时间步长

---

### 积分项

```python
self.integral += error * dt
```

👉 误差累计

---

### 微分项

```python
derivative = (error - self.prev_error) / dt
```

👉 误差变化速度

---

### 输出

```python
output = (kp * error + ki * integral + kd * derivative)
```

👉 PID 核心公式

---

## 4. 初始化系统状态

```python
x = 0.05
v = 0.0
```

👉 小球初始偏离中心

---

```python
theta = 0.0
```

👉 板子初始水平

---

## 5. 创建两个 PID

```python
pid_position = PID(5.0, 0.0, 2.0)
pid_angle = PID(20.0, 0.0, 1.0)
```

👉 外环（慢）+ 内环（快）

---

## 6. 主循环（核心）

```python
for i in range(steps):
```

每一步代表：0.01 秒

---

## 7. 外环：位置控制

```python
x_error = x_target - x
theta_ref = pid_position.update(x_error, dt)
```

👉 位置误差 → 目标角度

---

## 8. 内环：角度控制

```python
theta_error = theta_ref - theta
motor_cmd = pid_angle.update(theta_error, dt)
```

👉 角度误差 → 电机控制

---

## 9. 电机模型（简化）

```python
theta += motor_cmd * dt
```

👉 假设电机“直接控制角度”

（真实系统会更复杂）

---

## 10. 小球动力学

```python
a = g * theta
v += a * dt
x += v * dt
```

👉 三步：

1. 加速度
2. 速度
3. 位置

---

## 11. 打印结果

```python
print(x, v, theta)
```

👉 观察系统是否稳定

# 七、运行后你应该看到什么？

开始：

* x ≠ 0

然后：

👉 x 慢慢回到 0
👉 会有轻微振荡（正常）

# 八、调参入门口诀

## 先调内环（角度）

* kp ↑ → 更快
* kd ↑ → 更稳

---

## 再调外环（位置）

* kp ↑ → 拉回更快
* kd ↑ → 减少振荡
